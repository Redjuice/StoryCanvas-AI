# StoryCanvas AI Web Spec

## Why

为AI绘本生成器提供响应式Web前端，支持PC端和移动端访问，实现用户友好的绘本创作体验。

## What Changes

- 响应式布局（PC/移动端不同样式）
- 用户认证页面（登录/注册）
- 首页（功能展示/导航）
- 故事创作页面（主题输入/参数选择）
- 绘本预览页面（翻页/编辑）
- 我的书架（作品管理）
- 个人中心（用户信息）

## Impact

- 受影响系统: Vue3 + Vite前端项目
- UI框架: Tailwind CSS
- 状态管理: Pinia
- 路由: Vue Router

## ADDED Requirements

### Requirement: 响应式布局

系统 SHALL 根据屏幕宽度自动切换PC端和移动端布局。

#### Scenario: 移动端布局 (< 768px)

- **WHEN** 屏幕宽度 < 768px
- **THEN** 显示移动端优化布局
  - 底部导航栏
  - 单列布局
  - 紧凑间距

#### Scenario: PC端布局 (>= 768px)

- **WHEN** 屏幕宽度 >= 768px
- **THEN** 显示PC端优化布局
  - 顶部导航栏
  - 多列布局
  - 宽松间距

### Requirement: 认证页面

系统 SHALL 提供登录、注册和忘记密码页面。

#### Scenario: 登录页面

- **WHEN** 访问 /auth/login
- **THEN** 显示登录表单
  - 邮箱输入
  - 密码输入（可见性切换）
  - 登录按钮
  - 切换到注册链接
  - 忘记密码链接
  - 微信登录按钮

#### Scenario: 注册页面

- **WHEN** 访问 /auth/register
- **THEN** 显示注册表单
  - 邮箱输入
  - 密码输入
  - 确认密码输入
  - 注册按钮
  - 切换到登录链接
  - 微信注册按钮

#### Scenario: 忘记密码页面

- **WHEN** 访问 /auth/forgot-password
- **THEN** 显示忘记密码表单
  - 邮箱输入
  - 发送验证码按钮
  - 验证码输入
  - 新密码输入
  - 确认新密码输入
  - 重置密码按钮
  - 返回登录链接

### Requirement: Toast 提示组件

系统 SHALL 提供全局 Toast 提示组件用于显示操作反馈。

#### Scenario: 显示成功提示

- **WHEN** 操作成功（如登录成功、注册成功）
- **THEN** 显示绿色 Toast 提示
  - 成功图标
  - 提示消息
  - 自动消失（3秒）

#### Scenario: 显示错误提示

- **WHEN** 操作失败（如登录失败、表单验证错误）
- **THEN** 显示红色 Toast 提示
  - 错误图标
  - 错误消息
  - 自动消失（3秒）

#### Scenario: 显示警告提示

- **WHEN** 需要用户注意（如表单验证警告）
- **THEN** 显示橙色 Toast 提示
  - 警告图标
  - 警告消息
  - 自动消失（3秒）

#### Scenario: 显示信息提示

- **WHEN** 显示一般信息
- **THEN** 显示蓝色 Toast 提示
  - 信息图标
  - 提示消息
  - 自动消失（3秒）

### Requirement: 首页

系统 SHALL 根据设备类型展示不同的首页布局，Web端(>=768px)和H5端(<768px)有独立的UI设计。

#### Scenario: Web端首页 (>= 768px)

- **WHEN** 在PC端访问 /
- **THEN** 显示 Web端布局

**Hero 区块:**
- 品牌标签 "AI Powered Storytelling"
- 主标题 "让想象力插上翅膀，一键生成精美绘本"
- 副标题介绍产品价值
- CTA 按钮 "立即开启创作之旅"
- 右侧绘本示例图片网格（带旋转效果）

**艺术风格展示区块:**
- 区块标题 "探索无限艺术风格"
- 风格描述文字
- 左右滚动按钮
- Bento Grid 风格卡片展示（2大+2小布局）
  - 梦幻水彩 (Watercolor Dream)
  - 简约漫画 (Cartoon Pop)
  - 古典油画 (Oil Master)
- 卡片悬停放大效果

**功能特性区块:**
- 三列功能卡片
  - 秒级生成（闪电图标）
  - 风格定制（调色板图标）
  - 一键排版（书本图标）

**CTA 区块:**
- 渐变紫色背景
- 标题 "准备好讲述你的故事了吗？"
- 描述文字
- CTA 按钮
- 免费体验提示

**页脚:**
- 品牌 Logo 和版权信息
- 链接（联系我们/服务条款/隐私政策/社区指南）
- 社交媒体图标

#### Scenario: H5端首页 (< 768px)

- **WHEN** 在移动端访问 /
- **THEN** 显示 H5端布局

**顶部导航栏:**
- 汉堡菜单按钮
- 品牌名称 "StoryCanvas AI"
- 用户头像

**Hero 区块:**
- 全宽背景图片（4:5 比例）
- 渐变遮罩（从底部紫色渐变）
- 标题文字叠加在图片上
- CTA 按钮（全宽）

**功能特性区块:**
- 三列紧凑布局
- 图标+文字组合
- 圆角卡片设计

**探索画风区块:**
- 标题 "探索画风" + "查看全部"链接
- 横向滚动卡片列表
- 不对称偏移效果（第二张卡片向下偏移）
- 卡片可左右滑动

**AI推广横幅:**
- 左侧文字介绍
- 右侧装饰图片
- 标签展示（智能排版/连贯画风）

**页脚:**
- 简化的链接（Privacy/Terms）
- 版权信息

**底部导航栏:**
- 固定在底部
- 三个入口：首页/创作/书架
- 圆角胶囊设计
- 当前页面高亮显示

### Requirement: 故事创作页面

系统 SHALL 提供故事参数输入界面。

#### Scenario: Web端创作页 (>= 768px)

- **WHEN** 在PC端访问 /create
- **THEN** 显示 Web端布局
  - 左侧输入区域
    - 故事构思 textarea
    - 灵感示例按钮
    - 年龄段选择 (0-3/3-6/6-9/6+)
    - 视觉风格选择 (童趣动漫/温润水彩/复古素描/3D立体)
  - 右侧预览面板
    - 故事标题
    - 生成进度显示
    - 生成按钮

#### Scenario: H5端创作页 (< 768px)

- **WHEN** 在移动端访问 /create
- **THEN** 显示 H5端布局
  - 故事构思 textarea
  - 灵感快速启动按钮
  - 视觉风格网格选择
  - 生成状态显示
  - 底部悬浮生成按钮

### Requirement: 绘本预览页面

系统 SHALL 提供绘本预览和编辑功能。

#### Scenario: Web端预览页 (>= 768px)

- **WHEN** 在PC端访问 /preview/:id
- **THEN** 显示 Web端布局
  - Hero Header 区域
  - 左侧侧边栏
    - 故事详情（书名输入、画风显示）
    - PDF 导出按钮
  - 右侧故事网格
    - 页面卡片（图片 + 文字）
    - 悬停显示重新生成按钮
    - 文字编辑 textarea
  - 底部操作栏（保存草稿/立即发布）

#### Scenario: H5端预览页 (< 768px)

- **WHEN** 在移动端访问 /preview/:id
- **THEN** 显示 H5端布局
  - Hero 标题区域
  - 故事画布卡片列表
    - 图片 + 页面编号 + 文字
    - 编辑按钮
  - 底部操作栏（保存草稿/立即发布）
  - 底部导航栏

### Requirement: 我的书架

系统 SHALL 提供作品管理功能。

#### Scenario: Web端书架 (>= 768px)

- **WHEN** 在PC端访问 /bookshelf
- **THEN** 显示 Web端布局
  - Hero 区域
  - 搜索框
  - 书架网格（2-4列）
  - 创作新故事卡片
  - 图书卡片（封面、状态标签、标题）
  - 空状态显示

#### Scenario: H5端书架 (< 768px)

- **WHEN** 在移动端访问 /bookshelf
- **THEN** 显示 H5端布局
  - 搜索框
  - 水平筛选栏（全部故事/草稿箱/已发布）
  - 双列书架网格
  - 底部右侧悬浮按钮（创建新故事）
  - 底部导航栏

### Requirement: 个人中心

系统 SHALL 提供用户信息管理。

#### Scenario: 个人中心

- **WHEN** 访问 /profile (需要登录)
- **THEN** 显示
  - 用户头像
  - 用户昵称
  - 邮箱
  - 编辑资料按钮
  - 退出登录按钮
